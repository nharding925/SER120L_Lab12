# SER120L Flowers Lab

A simple interactive Java program where clicking the screen places flowers for my sophomore Software Engineering lab (SER120L).

## What It Does

A green canvas where each mouse click places a flower image at that location. It cycles through 10 different flower images with each click.

## How It Works

- `Flower` — stores an image and its position on the screen
- `FlowersPanel` — handles mouse clicks, loads 10 flower images, and renders all placed flowers
- `FlowersFrame` — sets up the 600x600 window
- `FlowersDemo` — entry point that launches the app

## Built With

- Java
- Java Swing
- Eclipse IDE
